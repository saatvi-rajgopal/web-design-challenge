Summary: City Latitude vs. ...

In this project, we used the OpenWeather API to pull data from the cities available, and compared the location (latitude/longitude) to other weather variables to see if we could find a specific trend.

We used Matplotlib to try and find different trends based on different factors. The factors that we compared with the latitudes/longitude include: temperature, cloudiness, wind speed, and humidity. This site provides the sources, the visualizations made from the data, and our observations.
